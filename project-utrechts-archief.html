Het Utrechts Archief

Leerjaar 2 · Module 6.1 2025–2026 Teamproject Schoolproject

Interactieve webapplicatie waarmee historische panoramafoto's van Utrecht doorzoekbaar en presenteerbaar worden gemaakt voor Het Utrechts Archief.

🤝

Samenwerking: Teamproject met merkbaar betere samenwerking dan eerdere projecten. Taken verdeeld op frontend, backend en data-verwerking.

Over dit project

Samen met mijn team hebben we een website/applicatie gemaakt voor Het Utrechts Archief. Historische panoramafoto's van Utrecht worden doorzoekbaar en visueel gepresenteerd via een interactieve viewer met 33 panelen.

Ik ben trots op de samenwerking binnen dit project — die liep merkbaar beter dan bij Annex Bios. De taakverdeling had bij de start duidelijker gekund, maar we hebben goed samengewerkt aan een professioneel eindresultaat.

Highlights

  • Panorama-viewer met 33 interactieve panelen
  • Intuïtieve paginaselectie en thumbnail-navigatie
  • Historische content toegankelijk gemaakt voor gebruikers
  • Professionele UI passend bij het archief
  • Sterke teamsamenwerking en taakverdeling

// Panorama-pagina laden op basis van selectie
function loadPanoramaPage(pageNumber) {
  const panel = document.querySelector('.panorama-panel');
  panel.src = `assets/panorama/page-${pageNumber}.jpg`;
  updateThumbnailActive(pageNumber);
}
